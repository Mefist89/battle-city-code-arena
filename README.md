# 🎮 CODETANK ARENA

**CODETANK ARENA** — это образовательная игровая веб-платформа, вдохновлённая классическими танковыми играми. В ней игроки управляют танками и учатся программированию, создавая скрипты для автоматизации действий, проходя миссии и соревнуясь на арене.

Текущая версия — **MVP (~v0.3)**.

---

## 🚀 Технологический стек

**Frontend:**
*   Фреймворк: **SvelteKit 2.57+** + **Svelte 5** (Runes)
*   Стилизация: **TailwindCSS v4**, кастомная дизайн-система (на базе shadcn-svelte)
*   Игровой рендер: **PixiJS 8.18+** (Canvas/WebGL)
*   Редактор кода: **CodeMirror 6** (с подсветкой Python)

**Backend:**
*   Фреймворк: **FastAPI** (Python)
*   Взаимодействие: **REST API** + **WebSockets** (для PvP и симуляции)

---

## 🗂️ Структура проекта

*   `/frontend/` — Пользовательский интерфейс, игровые экраны и рендеринг 2D-движка. Включает маршруты (`/game`, `/missions`, `/challenge`, `/pvp`).
*   `/backend/` — API-сервер. Занимается логикой симуляции сражений, проверкой написанного игроком кода (в песочнице) и управлением многопользовательскими сессиями.
*   `technical_spec.md` — Технические требования к проекту.

## 🛠️ Запуск проекта локально

Для запуска вам понадобятся установленные **Node.js 18+** и **Python 3.10+**.

> [!TIP]
> Рекомендуется запускать **сначала Backend**, а затем **Frontend**, чтобы клиентская часть могла сразу успешно подключиться к API и WebSockets.

### 1. Запуск Backend (FastAPI)

Откройте терминал и выполните следующие команды:

```bash
cd backend

# Активация виртуального окружения (если оно есть)
# На Windows:
.venv\Scripts\activate
# На Mac/Linux:
source .venv/bin/activate

# Если зависимостей еще нет, установите их:
pip install -r requirements.txt 
# (или напрямую: pip install fastapi uvicorn pydantic websockets anyio)

# Запуск сервера
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
Backend будет доступен по адресу `http://localhost:8000` (документация API — по адресу `/docs`).

### 2. Запуск Frontend (SvelteKit)

Откройте **новую вкладку** терминала:

```bash
cd frontend

# Установка всех зависимостей
npm install

# Запуск dev-сервера
npm run dev
```
Frontend запустится и будет доступен в браузере по адресу `http://localhost:5173`.

## 🎮 Игровые режимы

*   **Миссии (`/missions`)** — Поэтапное обучение и прохождение уровней через написание алгоритмов для танка.
*   **Арена (`/challenge`)** — Режим испытаний против нарастающих волн ИИ-противников.
*   **PvP Арена (`/pvp`)** — Сетевые сражения игроков в реальном времени.
